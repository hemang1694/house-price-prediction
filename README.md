# 🏠 House Price Prediction using Machine Learning

This project predicts house prices using machine learning regression models based on features such as area, number of rooms, location, and other property-related attributes.

## 🚀 Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## 📊 Models Used
- Linear Regression
- Ridge Regression
- Lasso Regression
- Gradient Boosting Regressor

## 🎯 Goal
To accurately predict house prices using regression techniques and compare multiple machine learning models.

## 📈 Best Model
Gradient Boosting performed best with higher accuracy and lower error.
